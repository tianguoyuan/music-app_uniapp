# 小程序的标识

目前有以下 `9` 种小程序标识，对应小程序平台类型如下：

| 类型         | 标识        |
| ------------ | ----------- |
| 微信小程序   | mp-weixin   |
| 支付宝小程序 | mp-alipay   |
| 抖音小程序   | mp-toutiao  |
| 飞书小程序   | mp-lark     |
| QQ小程序     | mp-qq       |
| 京东小程序   | mp-jd       |
| 小红书小程序 | mp-xhs      |
| 百度小程序   | mp-baidu    |
| 快手小程序   | mp-kuaishou |

> 注意： `mp-toutiao` 就是抖音小程序，其他的都很好辨别。
